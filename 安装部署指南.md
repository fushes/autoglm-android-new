# 智谱AutoGLM小米手机本地部署指南

## 项目概述

这是一个基于智谱AutoGLM开源模型的Android应用，可以在小米手机上实现本地AI助手功能，支持跨应用操作和智能自动化。

## 应用特性

- **本地AI模型**: 集成智谱AutoGLM 9B参数视觉-语言模型
- **跨应用操作**: 支持自动点击、滑动、输入等操作
- **屏幕分析**: 实时分析屏幕内容并智能响应
- **离线运行**: 所有AI推理在设备本地完成，无需网络连接
- **权限管理**: 完整的权限配置和安全保障

## 安装前准备

### 1. 环境要求
- 小米手机（Android 8.0+）
- 至少4GB可用存储空间（用于模型文件）
- 稳定的网络连接（首次下载模型）

### 2. 权限配置
安装前请确保手机允许安装未知来源应用：
- 设置 > 安全 > 未知来源应用安装 > 允许

## 安装步骤

### 方法一：使用预构建APK（推荐）

1. **下载APK文件**
   - 从项目仓库下载 `AutoGLM-debug.apk` 文件
   - 或通过构建脚本生成APK

2. **传输到手机**
   - 通过USB数据线、蓝牙或云存储将APK传输到手机

3. **安装应用**
   - 在手机上找到APK文件并点击安装
   - 按照提示完成安装过程

### 方法二：使用Android Studio构建

1. **环境准备**
   - 安装Android Studio 2022.3+ 
   - 安装Android SDK 34
   - 安装Java Development Kit 8+

2. **导入项目**
   ```bash
   # 克隆项目
   git clone <项目地址>
   
   # 在Android Studio中打开项目
   File > Open > 选择项目目录
   ```

3. **构建APK**
   - Build > Build Bundle(s) / APK(s) > Build APK(s)
   - 或使用构建脚本：`./build_apk.sh`

## 首次使用配置

### 1. 权限授予
安装完成后首次启动应用，需要授予以下权限：

- **存储权限**: 用于下载和存储模型文件
- **无障碍服务**: 用于跨应用操作
- **屏幕捕获**: 用于分析屏幕内容
- **悬浮窗权限**: 用于显示AI助手界面
- **摄像头权限**: 用于图像识别（可选）
- **音频录制权限**: 用于语音交互（可选）

### 2. 模型下载

1. **启动应用**
   - 打开"智谱AutoGLM"应用
   - 点击"下载模型"按钮

2. **下载过程**
   - 等待模型下载完成（约3-5分钟，取决于网络速度）
   - 下载进度会实时显示
   - 完成后"启动服务"按钮将变为可用

3. **验证模型**
   - 应用会自动验证模型文件的完整性
   - 验证通过后即可开始使用

## 使用说明

### 1. 启动AI服务
- 点击"启动服务"按钮
- 等待服务初始化完成
- 状态显示"服务运行中"

### 2. 输入任务
- 在任务输入框中输入要执行的任务
- 例如："打开微信并发送消息"
- 点击"执行任务"按钮

### 3. 查看日志
- 执行过程会实时显示在日志区域
- 可以查看AI的分析和操作步骤

### 4. 停止服务
- 点击"停止服务"按钮
- 服务停止后释放系统资源

## 故障排除

### 常见问题

1. **应用无法安装**
   - 检查是否允许未知来源应用安装
   - 确保APK文件完整无损

2. **权限被拒绝**
   - 前往设置 > 应用管理 > 智谱AutoGLM > 权限管理
   - 手动开启所有必要权限

3. **模型下载失败**
   - 检查网络连接
   - 确保存储空间充足
   - 重新下载模型

4. **服务无法启动**
   - 检查无障碍服务是否开启
   - 重启应用或手机

### 性能优化建议

1. **关闭后台应用**
   - 使用前关闭不必要的后台应用
   - 释放系统资源

2. **定期清理缓存**
   - 定期清理应用缓存
   - 优化运行性能

3. **避免同时运行**
   - 避免同时运行多个AI应用
   - 确保系统资源充足
